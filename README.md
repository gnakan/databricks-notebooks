# databricks-notebooks

Practitioner notebooks for Databricks experiments, by Gary Nakanelua (Databricks MVP).

These notebooks accompany published experiments and articles. Each is designed to be imported directly into Databricks and run.

## Contents

- **experiments/** — setup and data generation notebooks for practitioner experiments; each folder includes the raw pipeline output reviewed in the companion article

## How to import a notebook

1. Download the `.py` file
2. In your Databricks workspace: **Workspace → Import**
3. Select the downloaded file
4. Update any catalog, schema, or volume values at the top to match your environment
5. Attach to a cluster or use serverless compute and run
